# D42V110F12 Voltage Regulator Pin Summary

## Pin Descriptions

**VIN** — Input power pin. Accepts 3.5V–60V, in this application connected to the 24V vehicle supply. Requires bulk and decoupling capacitance at the pin.

**GND** — Ground reference. All grounds on the regulator are common and connect to the board ground plane.

**VOUT** — Regulated 12V output. Supplies power to the solenoid driver circuitry. Requires bulk output capacitance at the pin.

**PG (Power Good)** — Open-drain output that is pulled low when the output voltage falls outside of the acceptable range. Used as a fault indicator. Requires an external 100kΩ pull-up resistor to the logic voltage rail.

**ENA (Enable)** — Optional enable input. Pulled high internally through 1MΩ so the regulator runs by default. Can be driven low by a PLC signal to shut the regulator down and put it into a low power state.

**PFM (Mode Select)** — Selects between PFM and FPWM switching modes. Left floating in this design to use the default PFM mode, which dynamically adjusts switching frequency to maximize efficiency across varying load conditions.

**ENB** — Not used. This pin only has effect in FPWM mode and must not be used in the default PFM mode. Left unconnected.
